# OrientNet Site (Next.js)

## Setup
1. Install deps
```bash
npm install
```

2. Copy env
```bash
copy .env.example .env
```
(Or create `.env` manually.)

3. Run dev
```bash
npm run dev
```

Open: http://localhost:3000

## Routes
- `/` Home
- `/packages` Plans
- `/checkout?plan=turbo` Checkout (manual + PayPal)
